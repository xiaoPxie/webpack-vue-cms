# webpack-vue-cms
## 使用 webpack4 + vue2 + element-ui 搭建

> 另外用到了以下插件<br>
> ① vue-router： vue官方router<Br>
> ② vuex： 状态管理<Br>
> ③ mockjs： 生成随机数据，拦截ajax请求，可作为后端api接口模拟的插件<Br>
> ④ axios： ajax请求方案<Br>
> ⑤ lodash： js工具库
